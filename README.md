## Paleta de Colores

Esta paleta está diseñada para complementar el color base `RGB(41, 34, 36)`, un tono oscuro y neutro que se adapta bien a temas sofisticados y de bajo contraste visual. La paleta incluye tonos complementarios y de acento que aportan calidez, contraste y equilibrio al diseño.

### Color Base
- **Principal**: `RGB(41, 34, 36)`  
  Un tono oscuro y elegante ideal como color de fondo o base para otros elementos.

### Colores Complementarios
- **Rojo Oscuro**: `RGB(136, 49, 50)`  
  Añade calidez y contraste fuerte sin sobrecargar.
  
- **Crema Claro**: `RGB(230, 216, 207)`  
  Suave y claro, perfecto para fondos que necesitan resaltar sutilmente sobre el color base.

- **Gris Azulado Claro**: `RGB(183, 193, 200)`  
  Un tono fresco y suave, ideal para áreas secundarias o elementos de menor importancia.

- **Verde Oliva Oscuro**: `RGB(89, 94, 74)`  
  Un tono terroso que aporta equilibrio y un toque natural.

- **Azul Acero**: `RGB(102, 108, 124)`  
  Un tono frío que complementa el color base y ofrece elegancia sin competir visualmente.

### Colores de Acento
- **Mostaza**: `RGB(204, 173, 96)`  
  Acento cálido y llamativo que se destaca sobre el color base.

- **Terracota**: `RGB(189, 110, 84)`  
  Tono cálido y terroso que añade contraste sin ser demasiado brillante.

- **Blanco Humo**: `RGB(245, 245, 245)`  
  Perfecto para el texto o áreas que requieran mucha claridad y contraste con el fondo oscuro.

---

Esta paleta proporciona una combinación armónica de tonos oscuros y claros, ideales para interfaces elegantes y de bajo contraste, además de colores de acento que ayudan a resaltar elementos específicos.
